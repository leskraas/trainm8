Strength authoring as tokens

How a strength step reads and edits inside the sentence: the exercise (searchable combobox), the compact set notation (Back squat 5 × 5 @ 80 kg), the set list (kind · reps/secs · kg xor %1RM), and audit gap G10 — adding rest between sets to a step that has none. Three variants differ on where set editing happens and where the rest affordance lives. Try: change the squat's load, vary the deadlift's third set, give the plank a rest between sets.
